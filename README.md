# CRM Dashboard

Full-stack CRM built with Spring Boot 3.0 and React 18.

## Tech Stack

- **Backend:** Java 17, Spring Boot 3.0.2, Spring Security 6, PostgreSQL
- **Frontend:** React 18, TypeScript, Recharts, Tailwind CSS
- **Infrastructure:** Docker, Docker Compose

## Features

- Customer management (CRUD)
- Deal pipeline with Kanban board
- Dashboard with revenue charts and KPIs
- Activity tracking
- Search and filtering

## Getting Started

```bash
docker-compose up -d
```

- Backend: http://localhost:8080
- Frontend: http://localhost:3000
